# 毛南族文化展示网站

这是一个展示毛南族文化的网站，包含以下功能：
- 毛南族文化介绍
- 非物质文化遗产展示
- 文化搜索功能
- 智能问答系统

## 技术栈
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## 本地开发
1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 http://localhost:3000

## 部署
本项目使用 Vercel 进行部署，访问 https://vercel.com 进行部署。

## 环境要求
- Node.js 18.0.0 或更高版本
- npm 9.0.0 或更高版本
