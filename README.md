# MAONAN CULTURAL ARCHIVE

## Where Heritage Meets Interactive Web Storytelling

Maonan Cultural Archive is an immersive Next.js platform designed to preserve, present, and reimagine Maonan ethnic culture for the digital era.  
It blends visual storytelling, cultural knowledge, and interactive exploration into a modern multilingual-ready experience.

---

## Vision

To transform cultural preservation from static records into living digital experiences.

This project explores a core idea:

> Culture is not only something to read about.  
> It should be something you can navigate, feel, search, and converse with.

---

## Experience Highlights

### Cinematic Landing Experience
- Full-screen visual sections with layered overlays
- Carousel-driven hero storytelling
- Scroll-triggered reveal effects and smooth transitions

### Cultural Knowledge Atlas
- Structured cultural chapters (history, architecture, agriculture, language, religion, clothing, food)
- In-page table of contents with smooth section navigation
- Rich typography and image-forward presentation

### Intangible Heritage Hub
- Curated heritage cards with category levels (national, regional, local)
- Dedicated pages for official heritage listings
- Data-oriented layouts for clear cultural reference

### Smart Navigation Layer
- Global cultural navbar with utility shortcuts
- Integrated time context display (Gregorian + Lunar date)
- Seasonal/solar-term-inspired visual cues

### AI Conversation Prototype
- Built-in cultural Q&A interface
- Chat-style UI architecture ready for model integration
- Designed for future retrieval-augmented cultural assistance

### Search Foundation
- Search page scaffold prepared for indexing and semantic retrieval
- UX-ready query input and result layout for rapid feature extension

---

## Technology Stack

- Next.js 14 (App Router)
- React 18 + TypeScript
- Tailwind CSS
- Font Awesome Icons
- React Slick (immersive carousel system)
- Moment.js + SolarLunar (calendar intelligence)

---

## Project Structure Snapshot

- app/: Route-based pages and interactive UI
- app/components/: Shared navigation and reusable view logic
- public/images/: Visual identity and cultural assets
- types/: Custom type declarations (including solar-lunar typing)

---

## Why This Project Matters

Most cultural websites are static archives.  
This project aims to become a digital cultural interface:

- Discoverable: users can explore by topic and visual context
- Explainable: content is structured and progressively navigable
- Extendable: AI and search modules are ready for deeper intelligence
- Preservable: tradition is translated into maintainable digital systems
